## 📋 Implementation Specs for Replit

### **What to build:**

**1. ESP32 Sends Capability Message on Connect**
```
When ESP32 connects, it sends ONCE:
DEVICE_INFO:esp32LOGIC|VERSION:1.0|COMPONENTS:MPU6050,ESC,TEMP_SENSOR|BAUD:115200
HEALTH:MPU6050=OK|ESC=OK|TEMP=FAIL

Then normal data stream:
X:0.5 Y:1.2 Z:-0.3 RPM:8000 TEMP:42.5
```

**2. Backend (Node.js/Python) Changes:**

**a) Parse Device Info Message**
- Listen for `DEVICE_INFO:` line on serial
- Extract: `components[]`, `version`, `health_status{}`
- Store in a **new REST endpoint**: `/api/device/capabilities`

**b) Flush stale simulator data**
```
When DEVICE_INFO received:
- Clear memStore completely
- Clear all historical data buffers
- Set isHardwareConnected = true
- Set isSimulatorMode = false
```

**c) Create device capability endpoint**
```
GET /api/device/capabilities
Response:
{
  "connected": true,
  "components": ["MPU6050", "ESC", "TEMP_SENSOR"],
  "health": {
    "MPU6050": "OK",
    "ESC": "OK",
    "TEMP_SENSOR": "FAIL"
  },
  "version": "1.0"
}
```

**3. Frontend (React) Changes:**

**a) Fetch capabilities on app load**
```javascript
const { data: capabilities } = useQuery(
  'device-capabilities',
  () => fetch('/api/device/capabilities').then(r => r.json())
);
```

**b) Clear React Query cache when hardware connects**
```javascript
useEffect(() => {
  if (capabilities?.connected) {
    queryClient.invalidateQueries(); // Clear ALL cache
    queryClient.removeQueries(); // Remove old simulator data
  }
}, [capabilities?.connected]);
```

**c) Conditionally render panels based on capabilities**
```javascript
return (
  <>
    {capabilities?.components.includes('MPU6050') && 
      <MPU6050Panel health={capabilities.health.MPU6050} />
    }
    {capabilities?.components.includes('ESC') && 
      <ESCPanel health={capabilities.health.ESC} />
    }
    {capabilities?.components.includes('TEMP_SENSOR') && 
      capabilities.health.TEMP_SENSOR === 'OK' ? 
        <TempPanel /> : <FailedSensorWarning />
    }
  </>
);
```

**d) Add WebSocket listener for real-time data**
```javascript
useEffect(() => {
  if (!capabilities?.connected) return; // Only if hardware connected
  
  socket.on('sensor:data', (data) => {
    updateCharts(data); // Live updates
  });
}, [capabilities?.connected]);
```
